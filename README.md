### Grid Tool
Add precise grid overlays to your images with custom layouts and DPI settings.

**Features:**
- Custom canvas dimensions with DPI control
- Adjustable grid rows and columns
- Multiple grid colors (13 options)
- Image positioning and scaling
- Maintain aspect ratio option
- Fit-to-canvas mode
- Export high-resolution images
- All processing done locally in your browser

### Creative Spark
Get random two-word prompts to spark inspiration and break through creative blocks.

**Features:**
- Random word combination generator
- Dynamic color backgrounds
- Keyboard shortcuts (Spacebar to generate)
- Simple, distraction-free interface
- Perfect for warm-up exercises and creative prompts

### Palette Maker
Generate beautiful color palettes from images or random selection.

**Features:**
- Generate random color palettes
- Extract colors from uploaded images
- Lock/unlock individual colors
- Add/remove colors (up to 10)
- Color picker with image sampling
- Edit colors manually
- Export palettes in HEX, RGB, or HSL format
- All image processing done locally
